## Password Generator

This project was made loosely by following a tutorial to help me learn about Web Dev  
Made with HTML, CSS, and plain JavaScript  
Published at: https://noumanamalik.github.io/Password-Generator/

### Features
- Allows the user to make to generate a password with the desired length
- There will always be at least 1 number and/or symbol if the box is checked
- The user can hover their cursor over the password for the copy button to show up which copies the password to the users clipboard
